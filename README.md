# Cluska8 — marketing site

Next.js + Tailwind landing page for [Cluska8](https://cluska8.com) (AI customer support agent). Deploy on [Vercel](https://vercel.com) (recommended).

## Local dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Demo page (dev / screen shares)

Password-protected UI that talks to the Cluska8 API (run backend on port 8000):

`/demo?key=cluska8demo`

The Next dev server rewrites `/backend/*` → `http://localhost:8000/*`.

## Build

```bash
npm run build
npm start
```
